# 风汐管理系统(fancyx-admin)

## 项目介绍

风汐管理系统，使用.NET9+React18构建的RBAC通用权限管理系统（支持按钮级别权限），支持多租户功能，简单易上手，不使用任何三方Admin框架，完全作者独立开发；旨在为个人、企业提供高效、美观的后台管理解决方案，为.NET+React后台方案添砖加瓦， 系统采用最新最稳定的技术栈，具有良好的扩展性和可维护性，支持快速定制开发。

**核心特点：**

* 支持多租户
* 按钮级别权限控制
* 简洁高效的用户界面
* 模块化的系统架构
* 可读性高代码结构
